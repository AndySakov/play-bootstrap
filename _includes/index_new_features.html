Play 2.6.12 and Bootstrap 4 support

The project has been split to give support for Play 2.6.12 and Bootstrap 4. So now there are 6 parallel versions depending on which version you need for Play Framework and Bootstrap:

  • 1.3-P26-B3: Built for Play 2.6.12 and Bootstrap 3.
  • 1.3-P26-B4: Built for Play 2.6.12 and Bootstrap 4.
  • 1.3-P25-B3: Built for Play 2.5.16 and Bootstrap 3.
  • 1.3-P25-B4: Built for Play 2.5.16 and Bootstrap 4.
  • 1.3-P24-B3: Built for Play 2.4.11 and Bootstrap 3.
  • 1.3-P24-B4: Built for Play 2.4.11 and Bootstrap 4.

Note! There is a selector on the topbar's right side for change between all versions.
