Horizontal Form

This page its an example to show how to use a horizontal form.

First of all, you need to declare the horizontal field constructor as implicit. You have to specify its column widths for the form-group.

@code { @@implicitFieldConstructor = @@{ b4.horizontal.fieldConstructor("col-md-2", "col-md-10") } }

Tip: You could create your own default horizontal field constructor and import it with something like:

@code { @@import b4.horizontal.default.fieldConstructor }

To do that, you only need to add a file with this content to your views folder (views/b4/horizontal/default/package.scala):

@code { package views.html.b4.horizontal package object default { implicit val fieldConstructor = views.html.b4.horizontal.fieldConstructor("col-md-2", "col-md-10") } }
